# Visualización Simple

Este script demuestra cómo generar datos aleatorios de una distribución normal con `numpy` y graficarlos rápidamente usando `matplotlib`.

Originalmente diseñado en Google Colab para inyectar imágenes en formato base64 directamente en el output de Markdown, este script ha sido adaptado para entornos locales guardando la imagen generada directamente en la carpeta como `referencia.png` y proporcionando una ventana gráfica interactiva para el usuario.

## Requisitos previos

Asegúrate de haber instalado las dependencias base en `../requirements.txt`:

```bash
pip install -r ../requirements.txt
```

## Uso

Simplemente ejecuta el script:

```bash
python main.py
```
